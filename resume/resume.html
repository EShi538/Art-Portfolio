Leadership/positions:

  1. Art and Tech (Based in SJSU under the support of SVCSI) Assistant director
    1. Past Projects: 
      1. Interactive mirror (2020-2021): A 10x10 array of mirror modules. Each module contained a tiltable mirror connected to a servo motor in a laser cut box and press fit components. On the top there is a camera that sends information to code that separates any humanoids in the camera frame from the background using Human detection AI algorithm with OpenCV. The code then grids out the camera frame into a 10x10 grid (so it is in parity with the actual mirror wall “dimensions”) and then determines a gray-scale value for each grid section. Each grid square’s gray-scale value is converted to an angle of which to tilt each grid square’s corresponding mirror module. All together, with all of the components working in harmony, the mirror wall can create patterns replicating the humanoids captured in the camera frame and their movements in real time. The tilting mirrors create this illusion by manipulating the lighting, creating darker and lighter grid squares to “draw out” the patterns. 
      2. Wings(2021-2022): We decided to do something more artistic and made a pair of retractable wings that could be strapped on humans and dogs alike. We cut and connected pieces of plywood to create the frame and then attached feathers on it in a nice looking manner that makes it look like a real pair of wings.
      3. Interactive mirror Part II (2022-2023): In this, we aimed to improve on the original prototype because the original mirror wall was functional, but very clunky and almost impossible to move around without significant effort expenditure. In the new version, we 3-D printed the frame, instead of laser cutting and press fitting the frame. We also used command strips to stick the frame to the board, rather than glue and wires. Furthermore, our wire management was a lot more organized, allowing for easy fixing. Finally, we attached the mirror wall to some wheels for easy transport.
      4. Interactive LED Board (Modular) (2023 Summer): In this project, we created an interactive LED board that is modular. Input was taken on a phone, where the user could draw something with their finger, and then the data was sent from the iphone to an arduino and then to a LED board that’s wired up to the arduino board. The LED would then mirror the pattern the user inputted on the phone. The colors are customizable and it can create pre-set patterns too. It also has a clear function to erase the LED board.
